# sysmem-filesystem-c

## Description
C systems-programming project modeling a block-based file-system structure with boot record, bitmap, entry table, root directory, and data sections.

Academic context: operating systems / systems programming project focused on low-level file-system layout and binary data structures.
## Tech Stack
- C
- Code::Blocks
- GCC
- Single-language project

## Structure
- `main.c`: main entry point or primary source file.
- `*.cbp`: Code::Blocks project configuration.
- Source files: `copied_main.c`, `main.c`.

## How to Run
gcc main.c -o sysmem && ./sysmem
